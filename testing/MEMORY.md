## 缺陷库

### 常见缺陷模式
1. **边界值遗漏** - 最大/最小值没测试
2. **空指针** - 前端没做非空校验
3. **权限绕过** - 没登录也能访问
4. **数据一致性** - 缓存和数据库不一致

### 容易漏测的场景
- 网络断开
- 并发操作
- 超时处理
- 特殊字符输入
