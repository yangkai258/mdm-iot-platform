# Sprint QA - 2026-03-29

## 任务：重构所有前端页面，适配 Arco Pro 标准风格

### 背景
当前前端（`mdm-project/frontend/src/views/`）有 60+ 个视图页面，每个页面布局、样式各异，严重不符合 Arco Design Pro 的标准风格。

### Arco Pro 标准页面结构
```vue
<template>
  <div class="container">
    <!-- 1. 面包屑 -->
    <Breadcrumb :items="['menu.xxx', 'menu.xxx.yyy']" />

    <!-- 2. 白色卡片 -->
    <a-card class="general-card" title="页面标题">
      
      <!-- 3. 搜索表单区（3列栅格） -->
      <a-row>
        <a-col :span="8">
          <a-form-item label="字段1">
            <a-input v-model="formModel.field1" placeholder="请输入" />
          </a-form-item>
        </a-col>
        <a-col :span="8">
          <a-form-item label="字段2">
            <a-select v-model="formModel.field2" :options="options" />
          </a-form-item>
        </a-col>
        <a-col :span="8">
          <a-form-item label="字段3">
            <a-range-picker v-model="formModel.field3" style="width: 100%" />
          </a-form-item>
        </a-col>
        <!-- 右侧：查询 + 重置按钮 -->
        <a-col :flex="'86px'" style="text-align: right">
          <a-space direction="vertical" :size="18">
            <a-button type="primary" @click="search">
              <template #icon><icon-search /></template>
              查询
            </a-button>
            <a-button @click="reset">
              <template #icon><icon-refresh /></template>
              重置
            </a-button>
          </a-space>
        </a-col>
      </a-row>

      <a-divider style="margin-top: 0" />

      <!-- 4. 操作栏（新建靠左，下载靠右） -->
      <a-row style="margin-bottom: 16px">
        <a-col :span="12">
          <a-space>
            <a-button type="primary">
              <template #icon><icon-plus /></template>
              新建
            </a-button>
          </a-space>
        </a-col>
        <a-col :span="12" style="display: flex; justify-content: end">
          <a-button>
            <template #icon><icon-download /></template>
            下载
          </a-button>
          <a-tooltip content="刷新">
            <div class="action-icon" @click="search"><icon-refresh size="18" /></div>
          </a-tooltip>
        </a-col>
      </a-row>

      <!-- 5. 数据表格 -->
      <a-table :columns="columns" :data="data" :pagination="pagination" row-key="id" />
    </a-card>
  </div>
</template>
```

### 样式规范
- 页面背景：`#F2F3F5`（arco 灰）
- 卡片：`a-card.general-card`，白色背景，圆角
- 面包屑组件：在 `src/components/Breadcrumb.vue`（需创建或从 Arco Pro 模板复制）
- 图标必须全局注册（从 `@arco-design/web-vue/es/icon` 导入）

### 必须导入的图标（全局注册方案）
在 `src/main.js` 中：
```js
import ArcoVueIcon from '@arco-design/web-vue/es/icon'
import '@arco-design/web-vue/dist/arco.css'
app.use(ArcoVueIcon)
```
