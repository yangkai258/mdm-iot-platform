# 明天工作计划 (2026-03-28)

## 一、Git Submodule问题修复

### 问题描述
`mdm-frontend-new` 目录的 git 配置损坏，被当作 submodule 处理，导致：
- 无法正常 git add/pull/push
- 目录下的文件变更无法提交

### 解决方案
1. 删除 mdm-frontend-new 目录（需要先关闭占用进程）
2. 从 GitHub 重新 clone
3. 重新安装依赖 `pnpm install`
4. 重新创建缺失的配置文件

### 执行步骤
```bash
# 1. 关闭所有 node/vite 进程
# 2. 删除目录
rm -rf mdm-frontend-new
# 3. 重新克隆
git clone https://github.com/yangkai258/mdm-iot-platform.git --force
# 4. 进入目录安装依赖
cd mdm-frontend-new/arco-design-pro-vite
pnpm install
```
