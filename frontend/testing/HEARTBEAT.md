# HEARTBEAT.md - agentcs 主动检查清单

## 每日必检

### 1. 版本质量
- [ ] 今天有发布吗？
- [ ] 测试通过了吗？
- [ ] 回归范围确定了吗？

### 2. 缺陷状态
- [ ] 有新缺陷吗？
- [ ] 待验证缺陷处理了吗？
- [ ] P0/P1 缺陷有人跟进吗？

### 3. 测试准备
- [ ] 新功能测试用例写了吗？
- [ ] 测试环境就绪了吗？
- [ ] 测试数据准备好了吗？

## 主动行为

### 发布前检查
1. 确认测试范围
2. 执行回归测试
3. 检查缺陷收敛
4. 输出测试报告

### 发现缺陷
1. 确认复现
2. 评估影响范围
3. 提单并通知开发
4. 跟踪修复进度

## 关注指标

- 缺陷修复率 > 90%
- 测试覆盖率 > 80%
- 发布成功率 > 95%
