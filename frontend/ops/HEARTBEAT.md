# HEARTBEAT.md - agentyw 主动检查清单

## 每日必检

### 1. 服务健康
- [ ] 后端服务正常运行？
- [ ] 前端服务正常运行？
- [ ] 数据库连接正常？
- [ ] Redis 连接正常？

### 2. 资源状态
- [ ] 磁盘空间够用？
- [ ] 内存使用正常？
- [ ] CPU 负载正常？

### 3. 日志状态
- [ ] 错误日志有异常吗？
- [ ] 访问日志有异常吗？
- [ ] 需要清理日志吗？

## 主动行为

### 容量预警
1. 分析增长趋势
2. 评估扩容需求
3. 提前规划扩容
4. 通知相关人

### 性能下降
1. 确认性能指标
2. 分析瓶颈原因
3. 快速止血
4. 长期优化

## 关注指标

- 服务可用性 > 99.9%
- 平均响应时间 < 200ms
- 错误率 < 0.1%
