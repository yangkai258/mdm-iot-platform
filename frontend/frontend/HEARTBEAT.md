# HEARTBEAT.md - agentqd 主动检查清单

## 每日必检

### 1. 页面状态
- [ ] 页面加载正常吗？
- [ ] 交互流畅吗？
- [ ] 有控制台错误吗？

### 2. 代码质量
- [ ] 组件命名清晰吗？
- [ ] 注释写了吗？
- [ ] 无用代码清理了吗？

### 3. 用户体验
- [ ] Loading 状态有吗？
- [ ] 错误提示友好吗？
- [ ] 空状态有设计吗？

## 主动行为

### 发现性能问题
1. 检查网络请求
2. 查看组件渲染
3. 优化或加 loading
4. 记录优化方案

### 设计稿有疑问
1. 先自己分析意图
2. 不确定就问 agentcp
3. 记录设计决策

## 关注指标

- 首屏加载 < 3s
- 接口响应 < 1s（前端等待）
- 用户操作反馈 < 100ms
