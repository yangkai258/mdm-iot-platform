## 构建工具

### Vite
```bash
npm create vite@latest
npm run dev      # 开发
npm run build    # 构建
npm run preview  # 预览
```

### 常用命令
```bash
npm install          # 安装
npm update           # 更新
npm run lint         # 检查
npm run format       # 格式化
```
