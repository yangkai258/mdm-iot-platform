## 1. жҰӮиҝ°

OTAпјҲOver-The-Airпјүеӣәд»¶еҚҮзә§жҳҜ MDM дёӯеҸ°зҡ„ж ёеҝғиғҪеҠӣпјҢж”ҜжҢҒ M5Stack и®ҫеӨҮиҝңзЁӢйқҷй»ҳеҚҮзә§еӣәд»¶пјҢж”ҜжҢҒзҒ°еәҰеҸ‘еёғгҖҒиҝӣеәҰиҝҪиёӘе’ҢејӮеёёиҮӘеҠЁеӣһж»ҡгҖӮ

**дёҡеҠЎзӣ®ж Үпјҡ**
- еӣәд»¶еҢ…е®Ңж•ҙз”ҹе‘Ҫе‘Ёжңҹз®ЎзҗҶ
- ж”ҜжҢҒе…ЁйҮҸ/зҷҫеҲҶжҜ”/зҷҪеҗҚеҚ•зҒ°еәҰзӯ–з•Ҙ
- еҚҮзә§иҝӣеәҰе®һж—¶иҝҪиёӘ
- жҲҗеҠҹзҺҮдҪҺдәҺйҳҲеҖјиҮӘеҠЁжҡӮеҒң

## 4. жҺҘеҸЈе®ҡд№ү

### 4.1 еҲӣе»әеӣәд»¶еҢ…

```
POST /api/v1/ota/packages
```

**иҜ·жұӮдҪ“пјҡ** name, version, hardware_model, file_size, file_url, file_md5, is_mandatory, allow_downgrade, release_notes, created_by

**е“Қеә”пјҡ** еӣәд»¶еҢ…е®Ңж•ҙдҝЎжҒҜеҗ«id/is_active/created_at

### 4.2 еӣәд»¶еҢ…еҲ—иЎЁ

```
GET /api/v1/ota/packages
```

**Query:** hardware_model, page, page_size

**е“Қеә”пјҡ** еӣәд»¶еҢ…еҲҶйЎөеҲ—иЎЁ

### 4.3 еҲӣе»әйғЁзҪІд»»еҠЎ

```
POST /api/v1/ota/deployments
```

**иҜ·жұӮдҪ“пјҡ**
```json
{
  "name": "v1.3.0 зҒ°еәҰеҸ‘еёғ30%",
  "package_id": 1,
  "hardware_model": "CoreS3_Rover",
  "strategy_type": "percentage",
  "strategy_config": { "percentage": 30 },
  "pause_on_failure_threshold": 20.0,
  "scheduled_at": "2026-03-20T12:00:00Z",
  "created_by": "admin"
}
```

**е“Қеә”пјҡ** deploymentеҗ«status=pending, target_device_count, pending_count

### 4.4 йғЁзҪІд»»еҠЎеҲ—иЎЁ

```
GET /api/v1/ota/deployments
```

**е“Қеә”пјҡ** йғЁзҪІд»»еҠЎеҲҶйЎөеҲ—иЎЁпјҢеҗ«success_rate=(success_count/(success_count+failed_count))

### 4.5 жҡӮеҒң/жҒўеӨҚ/еҸ–ж¶ҲйғЁзҪІд»»еҠЎ

```
POST /api/v1/ota/deployments/:id/pause
POST /api/v1/ota/deployments/:id/resume
POST /api/v1/ota/deployments/:id/cancel
```

**е“Қеә”пјҡ** жӣҙж–°еҗҺзҡ„statusе’Ңpause_reason

### 4.6 и®ҫеӨҮжЈҖжҹҘOTA

```
GET /api/v1/ota/devices/:device_id/check
```

**е“Қеә”(жңүжӣҙж–°)пјҡ** has_update=true, current_version, latest_version, package(еҗ«file_url/file_md5/is_mandatory)

**е“Қеә”(ж— жӣҙж–°)пјҡ** has_update=false, message="еҪ“еүҚе·ІжҳҜжңҖж–°зүҲжң¬"

### 4.7 и®ҫеӨҮдёҠжҠҘOTAиҝӣеәҰ

```
POST /api/v1/ota/devices/:device_id/report
```

**иҜ·жұӮдҪ“пјҡ** { "deployment_id": 1, "ota_status": "downloading", "progress_percent": 45, "ota_message": "жӯЈеңЁдёӢиҪҪ..." }

**е“Қеә”пјҡ** жӣҙж–°еҗҺзҡ„ota_status/progress_percent

### 4.8 еҚҮзә§иҝӣеәҰеҲ—иЎЁ

```
GET /api/v1/ota/deployments/:id/progress
```

**е“Қеә”пјҡ** summary(еҗ«pending/running/success/failed/success_rate) + еҲҶйЎөlist
