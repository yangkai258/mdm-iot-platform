- **Swagger** - 自动生成 API 文档
- **Apifox** - API 设计+文档+测试
- **Postman** - API 测试

### 原型设计
- **Figma** - 协作设计
- **Excalidraw** - 手绘风格流程图
- **draw.io** - 技术架构图

### 需求管理
- **Notion** - 文档管理
- **飞书** - 团队协作
- **Tower** - 任务管理

## 常用模板

### 需求文档模板
```markdown
# 功能名称
## 背景
## 目标
## 功能清单
## 用户故事
## 原型链接

```

### 用户故事模板
```
作为 [角色]
我想要 [功能]
以便 [价值]
验收标准:
- [条件1]
- [条件2]
```
