## 附录A：术语表

### A.1 多租户相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 租户 | Tenant | 在多租户系统中注册的独立企业/组织实体，拥有独立的用户、数据和配置 |
| 租户编码 | Tenant Code | 租户在系统中的唯一标识符，用于登录和 API 路由 |
| 超管 | Platform Admin / Super Admin | 平台运营方管理员，可管理所有租户 |
| 租户管理员 | Tenant Admin | 租户内部最高权限用户，可管理本单位用户和业务 |
| 套餐 | Plan | 租户订阅的服务等级，决定功能范围和配额上限 |
| 配额 | Quota | 租户可使用的资源上限，如设备数量、用户数量等 |
| 租户隔离 | Tenant Isolation | 不同租户之间的数据完全隔离，跨租户访问被禁止 |

### A.2 设备管理相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 设备 | Device | 通过 MDM 系统管理的智能硬件设备（如 M5Stack） |
| 设备码 | Device Code / SN | 设备唯一序列号，用于设备注册和绑定 |
| 设备影子 | Device Shadow | 存储设备实时状态的数据结构，保存在 Redis 中，TTL 90s |
| 心跳 | Heartbeat | 设备定期上报的在线状态消息，用于判断设备是否在线 |
| 绑定 | Bind | 将设备与用户/会员建立关联关系的过程 |
| OTA | Over-The-Air | 无线固件升级，通过网络向设备推送新版本固件 |
| 固件 | Firmware | 嵌入式设备上运行的软件程序 |
| 设备状态 | Device Status | online（在线）/ offline（离线）/ unknown（未知） |

### A.3 权限与用户相关

| 术语 | 英文 | 定义 |
|------|------|------|
| RBAC | Role-Based Access Control | 基于角色的访问控制，通过角色分配权限 |
| 租户用户 | Tenant User | 属于特定租户的注册用户，受租户权限体系管理 |
| 菜单权限 | Menu Permission | 控制用户可见菜单项的权限 |
| 按钮权限 | Button Permission | 控制用户可见操作按钮的权限（如新建/编辑/删除） |
| 数据权限 | Data Permission | 控制用户可访问数据范围的权限（如部门数据隔离） |
| 部门 | Department | 租户内部组织结构，支持树形层级 |

### A.4 会员营销相关

| 术语 | 英文 | 定义 |
|------|------|------|
| 会员 | Member | 在商家注册的用户，可享受会员权益 |
| 会员等级 | Member Level | 会员的等级体系，如普通、银卡、金卡等 |
| 积分 | Points | 会员消费或行为积累的虚拟货币 |
| 标签 | Tag | 描述会员特征的关键字，如「高消费」「活跃用户」 |
| 优惠券 | Coupon | 可抵扣金额或享受折扣的电子凭证 |

### A.5 技术架构相关

| 术语 | 英文 | 定义 |
|------|------|------|
| MQTT | Message Queuing Telemetry Transport | 轻量级物联网消息传输协议 |
| JWT | JSON Web Token | 用于身份认证的 Token 标准 |
| GORM | Go ORM | Go 语言的 ORM 库 |
| Redis | Redis | 内存数据库，用于缓存和设备影子存储 |
| Schema | Database Schema | PostgreSQL 中的数据库对象集合 |
| UUID | Universally Unique Identifier | 通用唯一标识符 |
| API | Application Programming Interface | 应用程序编程接口 |
