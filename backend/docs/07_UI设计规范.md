# 07 UI设计规范

**版本号：** V2.2  
**更新日期：** 2026-03-21

---

## 二十一、UI设计规范

> **功能路径**：全局规范  
> **权限要求**：所有用户  
> **套餐要求**：所有套餐

### 21.1 按钮规范

#### 21.1.1 按钮命名规范

所有按钮必须使用「」双尖括号包裹中文名称，禁止使用英文或无包裹格式。

**正确示例：**
- 「搜索」「刷新」「新增用户」「编辑」「删除」「保存」「取消」「导出」「导入」

**错误示例：**
- ❌ `[搜索]`、`Search`、`search-btn`、`新增用户`

#### 21.1.2 按钮类型定义

| 按钮类型 | 视觉样式 | 使用场景 |
|----------|----------|----------|
| 主按钮 | 实心蓝色/品牌色 | 主要操作，如「保存」「提交」「通过」 |
| 次按钮 | 边框样式 | 次要操作，如「取消」「重置」「返回」 |
| 危险按钮 | 红色实心/边框 | 危险操作，如「删除」「禁用」「拒绝」 |
| 文字按钮 | 无边框 | 低强调操作，如「查看详情」「编辑」 |

#### 21.1.3 按钮权限要求

| 权限标识 | 说明 | 可见按钮 |
|----------|------|----------|
| all | 所有登录用户 | 搜索、刷新、查看详情 |
| tenant_admin | 租户管理员 | 含新建、编辑、删除、配置等全部写操作 |
| super_admin | 超级管理员 | 平台级管理，如删除租户、禁用用户 |

#### 21.1.4 按钮状态

| 状态 | 说明 | 视觉表现 |
|------|------|----------|
| normal | 默认状态 | 正常颜色，可点击 |
| hover | 悬停状态 | 颜色加深，显示手型光标 |
| loading | 加载状态 | 按钮禁用，显示加载图标 |
| disabled | 禁用状态 | 颜色变灰，不可点击 |

### 21.2 布局规范

#### 21.2.1 页面布局类型

| 布局类型 | 说明 | 适用页面 |
|----------|------|----------|
| 列表页 | 顶部操作栏 + 数据表格 | 用户列表、设备列表、门店列表 |
| 表单页 | 左侧标签页 + 右侧表单 | 系统配置、参数设置 |
| 详情页 | 顶部信息卡 + 下方Tab | 租户详情、用户详情 |
| 树+列表 | 左侧树形 + 右侧列表 | 组织架构、档案分类 |
| 看板页 | 拖拽卡片布局 | 流程审批、任务管理 |

#### 21.2.2 操作栏规范（ArcoDesign Pro V2.2 列表页布局）

每个列表页面必须遵循以下统一的操作栏布局规范：

**布局结构：**
```
┌──────────────────────────────────────────────────────────┐
│  工作台 / 组织管理 / 公司管理        (面包屑，左上角)     │
│                                                          │
│  🔍 搜索...                         (搜索框，左对齐，280px)│
│                                                          │
│  「新建」「导出」「刷新」            (按钮组，左对齐)     │
│──────────────────────────────────────────────────────────│
│                                                          │
│                        数据表格                          │
│                                                          │
└──────────────────────────────────────────────────────────┘
```

**布局要点：**
1. **面包屑**：页面最左上角，格式：首页 / 模块名 / 页面名
2. **搜索框**：面包屑下方，左对齐，宽度约 280px，支持关键字筛选
3. **按钮组**：搜索框同一行下方（换行），靠左排列，顺序为「新建」→「导出」→「刷新」
4. **数据表格**：按钮组下方，占据页面剩余空间

**规则：**
- 面包屑在最顶部，显示当前页面路径
- 搜索框位于面包屑下方，左对齐，宽度约 280px
- 主操作按钮：「新建」「编辑」「删除」靠左排列，间距 8px
- 辅助操作按钮：「批量导入」「导出」「🔄刷新」靠在一起
- 按钮之间间距：8px
- 按钮高度：32px（标准）或 28px（紧凑）

#### 21.2.3 表格布局规范

| 元素 | 规范 |
|------|------|
| 列宽 | 最小 80px，最大自适应，超长文字省略 |
| 行高 | 标准 48px，紧凑 36px |
| 斑马纹 | 可选，建议复杂表格启用 |
| 复选框 | 多选表格第一列，宽度 40px |
| 操作列 | 固定右侧，最小宽度 120px（V2.2改为点击名称打开详情） |
| 空状态 | 居中显示图标+文字，无"暂无数据"纯文字 |

### 21.3 交互规范

#### 21.3.1 删除操作二次确认

所有删除操作必须弹出确认对话框，显示待删除对象信息：

```
确认删除
确定要删除以下数据吗？此操作不可撤销。
• 租户：深圳市XX科技有限公司
「取消」  「确定删除」
```

**危险操作确认：**
- 删除租户（super_admin）
- 删除部门（含下级）
- 删除用户（含数据）
- 禁用租户管理员

#### 21.3.2 Toast 消息规范

| 消息类型 | 显示位置 | 持续时间 | 示例 |
|----------|----------|----------|------|
| success | 顶部居中 | 3秒 | "保存成功"、"删除成功" |
| error | 顶部居中 | 5秒 | "网络错误，请重试" |
| warning | 顶部居中 | 4秒 | "配额已满，升级套餐" |
| info | 顶部居中 | 3秒 | "数据已刷新" |

#### 21.3.3 空状态规范

页面无数据时显示空状态插画，禁止留白：

**各模块空状态文案参考：**

| 模块 | 空状态文案 |
|------|-----------|
| 租户列表 | 暂无任何租户，点击左上角「新建租户」开始使用 |
| 用户列表 | 暂无用户，点击左上角「新增用户」添加 |
| 设备列表 | 暂无设备，请先注册设备 |
| 门店列表 | 暂无门店，点击左上角「新建门店」添加 |
| 会员列表 | 暂无会员数据 |

#### 21.3.4 页面加载规范

| 状态 | 表现 |
|------|------|
| 首次加载 | 全页骨架屏（Skeleton），禁止纯 loading 动画 |
| 刷新数据 | 表格区域局部 loading 遮罩，原有数据不变 |
| 提交请求 | 按钮显示 loading 状态，禁止双击 |
| 切换Tab | 旧内容保持，新内容显示骨架屏 |

#### 21.3.5 表单校验规范

| 校验时机 | 表现 |
|----------|------|
| 失焦校验 | 红色边框 + 下方错误提示文字 |
| 提交校验 | 滚动到第一个错误处，红色边框闪烁 |
| 实时校验 | 输入后 500ms 触发，红色边框 |

### 21.4 弹窗规范

#### 21.4.1 弹窗类型选择（V2.2 新增详情弹窗）

| 弹窗类型 | 触发场景 | 尺寸 |
|----------|----------|------|
| 全屏模态（Modal） | 新建/编辑复杂表单（如新建租户、编辑用户） | 宽度 800~1200px，高度 80vh |
| 抽屉（Drawer） | 快速操作（如分配角色、筛选条件） | 宽度 400~600px |
| 确认对话框（Dialog） | 删除确认、状态切换确认 | 宽度 400px，固定居中 |
| 消息提示（Toast） | 操作结果反馈 | 居中浮现 |
| **详情弹窗** | 点击名称/id打开详情 | 宽度 600~800px，仅显示信息 |

#### 21.4.2 模态框规范

**新建/编辑模态（全屏模态）：**
- 右上角：「×」关闭按钮 + 「保存」+「取消」
- 内容区：表单主体，滚动
- 底部：固定在模态框底部，「取消」在左，「保存」在右

**抽屉规范：**
- 从右侧滑入
- 右上角：「×」关闭按钮
- 宽度：400px（筛选）、500px（详情）

**详情弹窗（V2.2 新增）：**
- 右上角：「×」关闭按钮
- 内容区：只读展示信息，滚动
- 底部：**仅显示「取消」按钮**，无保存/编辑按钮

### 21.5 响应式规范

#### 21.5.1 断点定义

| 断点 | 屏幕宽度 | 布局变化 |
|------|----------|----------|
| 超大屏 | ≥1920px | 表格列全展示 |
| 大屏 | 1440~1919px | 标准布局 |
| 中屏 | 1024~1439px | 操作栏换行 |
| 小屏 | 768~1023px | 表格水平滚动 |
| 平板 | <768px | 强制收起操作栏 |
