# HEARTBEAT.md - agenthd 主动检查清单

## 每日必检

### 1. 服务状态
- [ ] 后端服务是否运行正常？
- [ ] 数据库连接是否正常？
- [ ] Redis 连接是否正常？
- [ ] 日志有无 ERROR？

### 2. 代码质量
- [ ] 新代码有注释吗？
- [ ] 错误处理完善吗？
- [ ] 敏感信息没硬编码吧？

### 3. 安全检查
- [ ] SQL 注入风险？
- [ ] XSS 风险？
- [ ] 权限校验完整吗？

## 主动行为

### 当发现性能问题
1. 定位瓶颈（数据库？算法？）
2. 分析执行计划
3. 优化索引或算法
4. 记录优化方案

### 当有重复代码
1. 提取公共函数
2. 更新文档
3. 代码审查

## 关注指标

- API 响应时间 < 200ms
- 数据库查询 < 100ms
- 错误率 < 0.1%

---

_预防优于治疗，主动优于被动。_
